# Quota Buddy

[English](README_EN.md) · [MIT License](LICENSE)

Quota Buddy 是一款 Windows 小工具，在 Codex 官方宠物下方显示 5 小时和每周剩余额度及剩余重置次数。

## 使用

1. 打开 Codex 桌面版和官方宠物。
2. 双击 `启动额度伴侣-中文.cmd`。
3. 拖动标题区域可移动，拖动右下角可调整大小；右键可刷新或退出。

首次运行会添加 Windows 登录启动项。额度框会跟随宠物移动，并在宠物关闭时隐藏。

## 功能

- 显示剩余额度、恢复时间和可用重置次数。
- 根据窗口宽度自动调整布局。
- 额度降低时，状态灯会变色并加快呼吸动画。

呼吸灯按两项额度中的较低值显示：绿色为 50% 及以上，黄色为 21%–49%，红色为 6%–20%，深红色为 5% 及以下；数据不可用时为灰色。

## 隐私

程序只读取 Codex 的只读额度服务和本机运行记录，不读取、保存或传输账号密码、令牌及登录凭据，也不生成日志。

## 自检

双击 `运行自检.cmd`。显示“全部通过 / All tests passed”即表示运行正常。
