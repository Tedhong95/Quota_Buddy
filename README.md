# Quota Buddy

[English](README_EN.md) · [MIT License](LICENSE)

Quota Buddy 是一款 Windows 小工具，在 Codex 官方宠物下方显示每周剩余额度、可用重置次数、重置类型及到期时间。

## 使用

支持 Windows 10/11，需要已登录的最新版 GPT/Codex 桌面客户端，无需管理员权限或额外安装环境。

1. 下载并解压完整项目文件夹，放在一个固定位置。
2. 打开 GPT/Codex 桌面客户端。
3. 双击 `启动额度伴侣-中文.cmd`。
4. 拖动标题区域可移动；拖动右边缘或右下角可调整宽度。高度会在拖动停止后自动收紧到完整容纳内容且无留白的尺寸；右键可刷新或退出。

首次运行会添加 Windows 登录启动项。额度框会在宠物开启时位于宠物正下方并跟随移动；没有宠物时默认位于主屏幕右下角。

## 功能

- 显示每周剩余额度、恢复时间，以及可用重置的类型和到期时间。
- 根据窗口宽度自动调整布局。
- 根据内容自动收紧高度，避免缩放时发生遮挡或产生多余留白。
- 额度降低时，状态灯会变色并加快呼吸动画。

呼吸灯按每周剩余额度显示：绿色为 50% 及以上，黄色为 21%–49%，红色为 6%–20%，深红色为 5% 及以下；数据不可用时为灰色。

## 隐私

程序只读取 Codex 的只读额度服务和本机运行记录，不读取、保存或传输账号密码、令牌及登录凭据。发生读取或启动故障时，只在本机 `.codex` 文件夹保存不含登录凭据的诊断记录。

## 自检

双击 `运行自检.cmd`。显示“全部通过 / All tests passed”即表示运行正常。
